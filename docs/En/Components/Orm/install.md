# ORM
A new coroutine-safe ORM package from Easyswoole.
# Installation

Dependency

- swoole `>= 4.4.8`
- Easyswoole  `>=3.3.2` 
- mysqli > `2.x`

```php
composer require easyswoole/orm
```
